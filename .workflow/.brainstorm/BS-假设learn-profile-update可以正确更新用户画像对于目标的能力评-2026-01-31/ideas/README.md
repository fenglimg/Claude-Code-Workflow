# Ideas

This folder holds deep-dive writeups for selected ideas during the interactive refinement loop.

Suggested filename format:
- `phase-state-machine.md`
- `handoff-contract.md`
- `gemini-review-diff-protocol.md`
- `exa-enrich-verify.md`
- `schema-auto-heal.md`

Each deep-dive should include:
- Concept
- Minimal viable version
- Inputs/outputs (schema sketch)
- Risks & mitigations
- Success metrics
- Next implementation steps
